C1.1 Merknad fra stemmestyrene


 Støren — valgtingsstemmer ordinære:
 Velgere kan ha stemt bare ved fylkestingsvalg. Noen sedler uten stempel der velgere kan ha levert utenpå
 seddel.

 Soknedal — valgtingsstemmer ordinære:
 En stemmeseddel uten stempel, ser ut til å være brukt som omslag. Ikke alle stemte i kommunevalget.

 Singsås — valgtingsstemmer ordinære:
 En diff. på 2 stemmer ved kontrolltelling

 Budal — valgtingsstemmer ordinære:
 Kan være at noen har stemt bare på fylkestingsvalget.

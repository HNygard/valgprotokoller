Innsyn - valgprotokoll 2019

Oversender møtebøkkene for Fylkestings- og kommunestyrevalget i Holmestrand kommune 2019, som
etterpurt i epost 17.09.2019.

På dine spørsmål:
1. Ble foreløpig opptelling utført manuelt eller med dataprogram?
   Foreløpig opptelling ble utført manuelt i kretsene. Resultatet lagt inn i EVA.

2. Ble endelig opptelling utført manuelt eller med dataprogram?
   Endelig opptelling ble gjennomført ved skannersentralen i Horten. Resultatet lagt inn i EVA.

3. Hvor mange opptellinger og omtellinger ble foretatt totalt?
   Det ble gjort en skanning av endelig opptelling per kasse i Horten, ikke noen omtellinger.

4. Hvis flere enn foreløpig og endelig, hvordan ble disse utført (manuelt/data?) og hvorfor ble de utført?
   Var ikke aktuelt.

Håper at det svarene er det du etterspør, ta kontakt om det er noe som er uklart.
